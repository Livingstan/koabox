## About Koabox

Koabox is a boilerplate for koajs framework. JWT secured and non secured routes  

## What is included

- Koa 2
- Mongodb
- Koa Router
- JWT Secured router
- ES7 with Babel
- Nodemon

### Usage

```bash
$ npm install 

# Start application
$ npm start

```
Note: you have to start the mongod first.
